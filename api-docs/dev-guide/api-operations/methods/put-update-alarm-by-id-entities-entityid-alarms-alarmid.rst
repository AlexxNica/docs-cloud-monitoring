Update alarm by ID
^^^^^^^^^^^^^^^^^^
.. code::

    PUT /entities/{entityId}/alarms/{alarmId}

Update an alarm by using a valid set of attributes from the
`alarms attributes <http://docs.rackspace.com/cm/api/v1.0/cm-devguide/content/service-alarms.html>`__
table.
